Detailed Description

VanGenuchten - Parker relative permeability curves.

  • In this curves the relative permeabilities are found from:
  • krw(S) = kw0 sqrt(S) ( 1-( 1-S^(1/m) )^m )^2
  • krn(S) = kn0 sqrt(1-S) ( 1-S^(1/m) ) ^ 2m
