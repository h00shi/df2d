bool DuplData::operator< ( const DuplData &  y) const

Sorting operator.

As I mentioned in my thesis, at each node there can be multiple DuplDatas, but only one of them is the master. DuplData x will be less than y when y is a better candidate to be the master. The more the region's idx, the more the DuplData's priority.

Index of the DuplData.

The saturation and potential values are stored in the MData class. This index allows each DuplData to access its relevant saturation, mobilities, cappilary pressure, ... in the MData.
