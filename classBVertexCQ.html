Detailed Description

Class for boundary vertex with constant in(out)flow.

This will be the base class and other boundary vertex types will be inherited from it. The BVertex classes are responsible for imposing boundary conditions. For each pressure(potential) boundary condition there is a different BVertex class. However, The saturation boundary condition is only controlled using an if statement and an enum.

Note
To construct:
  • use constructor. (through Mesh)
  • add neighbours. (through Mesh)
  • check neighbours (through Mesh through Element)
  • constructBase (through Mesh)
To use:
  • call assemP before solving p equation (through solver)
  • call findQAll after solving p equation and assembling s eqution(through solver)
  • call assemS after calling findQAll (through solver)
  • use mGammaW and mGamma to update overall flux (through solver)
To destroy:
  • use delete over all list members
