template<CellType C>
const arma::mat & ElementPoly< C >::matKD ( )
inlinevirtual

find the D matrix.

Returns
the D matrix evaluated at the center point. D * P = Grad P.
Note
The retuned value is a reference value. It means that there is a static (much like global) arma::rowvec member of ElementBase class and calling this function will change that static member and return a reference value to it. So if your have previously called matJ, matB or matX and stored pointers to them, i.e.: 
arma::mat *J;
J = &element.matJ(xi,eta);
your pointers will point to new values after calling this function!
